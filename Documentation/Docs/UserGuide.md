## Getting Started

### Welcome to Swiff!

Swiff helps you track all your subscriptions, manage expenses, and never miss a payment. Whether you have 5 subscriptions or 50, Swiff keeps everything organized beautifully.

### First Launch

When you first open Swiff, you'll see:

1. **Home Screen** - Your financial dashboard
2. **Empty States** - Helpful prompts to add your first data
3. **Quick Actions** - Easy ways to get started

### Adding Your First Subscription

#### Method 1: Quick Add Button
1. Tap the **"+"** button at the top right of any screen
2. Select **"Add Subscription"**
3. Fill in the details:
   - **Name**: e.g., "Netflix"
   - **Price**: Monthly cost
   - **Billing Cycle**: How often you're charged
   - **Next Billing Date**: When it renews
   - **Category**: Entertainment, Productivity, etc.
4. Tap **"Save"**

That's it! Your subscription is now tracked.

#### Method 2: Quick Action Menu
1. Tap the floating **"+"** button (bottom right on Home screen)
2. Select from quick actions
3. Fill in details and save

### Adding Sample Data

Want to try Swiff without adding real data?

1. Go to **Settings**
2. Scroll to **Data Management**
3. Tap **"Add Sample Data"**
4. Swiff creates 5-10 sample subscriptions for you to explore
5. Delete sample data anytime from Settings

## Tips & Tricks

### Subscription Management

#### Track Free Trials
1. Add subscription as usual
2. Toggle "Free Trial" option
3. Set trial end date
4. Enable trial expiration reminders
5. Get alerted before conversion to paid

#### Price History
- Swiff automatically tracks when you change subscription prices
- View price history chart in subscription detail
- See percentage increases/decreases
- Get alerted to price increases

#### Shared Subscriptions
1. Add subscription normally
2. Tap "Share Subscription"
3. Select people to share with
4. Set split type:
   - Equal split
   - Custom amounts
   - Percentage-based
5. Swiff calculates who owes what

#### Pause vs Cancel
- **Pause**: Temporarily stop tracking (subscription still active)
- **Cancel**: Mark as cancelled (stop renewal reminders)
- **Delete**: Remove completely from Swiff

### Transaction Tips

#### Adding Receipts
1. Edit any transaction
2. Tap "Add Receipt"
3. Take photo or choose from library
4. Receipt is stored securely on device

#### Linking to Subscriptions
- Manually link transaction to subscription
- View all charges for a subscription
- Track actual spending vs expected cost

#### Bulk Operations
1. Tap **"Select"** in transaction list
2. Tap multiple transactions
3. Choose bulk action:
   - Delete all selected
   - Change category for all
   - Add tag to all
   - Export selected

### People & Sharing

#### iOS Contacts Integration

#### Settle Balances

- Combine multiple filters

## Troubleshooting

### Notifications Not Working

**Problem**: Not receiving renewal reminders or alerts.

**Solutions**:

1. **Check System Permission**
   - Settings > Notifications > Swiff
   - Ensure "Allow Notifications" is ON
   - Check notification style is set

2. **Check In-App Settings**
   - Open Swiff Settings
   - Ensure "Enable Notifications" is ON
   - Check specific notification types are enabled

3. **Verify Reminders Are Set**
   - Open subscription detail
   - Check "Reminder Settings"
   - Ensure reminder days are configured

4. **Send Test Notification**
   - Settings > Notifications > "Send Test Notification"
   - Should receive test immediately
   - If not, check system settings again

### Data Not Saving

**Problem**: Changes aren't persisting after closing app.

**Solutions**:

1. **Force Close and Reopen**
   - Swipe up from bottom to see open apps
   - Swipe Swiff up to close
   - Reopen Swiff

2. **Check Available Storage**
   - Settings > General > iPhone Storage
   - Ensure sufficient space available
   - Swiff needs at least 50MB free

3. **Restart Device**
   - Sometimes iOS caching issues occur
   - Full device restart can help

4. **Restore from Backup**
   - If data is corrupted
   - Use latest backup to restore

### Import Fails

**Problem**: Can't import backup or CSV file.

**Solutions**:

1. **Check File Format**
   - Backups: Must be .json
   - Exports: Must be .csv or .json
   - Ensure file isn't corrupted

2. **Verify File Source**
   - If from email, download first
   - Save to Files app
   - Then import from Files

3. **Check File Permissions**
   - Ensure Swiff has access to Files
   - Settings > Privacy > Files and Folders

### App Crashing

**Problem**: App closes unexpectedly.

**Solutions**:

1. **Update to Latest Version**
   - Check App Store for updates
   - Install any available updates

2. **Restart Device**
   - Hold power button
   - Slide to power off
   - Turn back on

3. **Reinstall App** (Last Resort)
   - Create backup first!
   - Delete app
   - Reinstall from App Store
   - Restore from backup

### Performance Issues

**Problem**: App is slow or laggy.

**Solutions**:

1. **Large Dataset**
   - If you have 1000+ transactions
   - Consider archiving old data
   - Export and delete transactions older than 2 years

2. **Clear Cache** (if available)
   - Settings > Data Management
   - Clear cache (doesn't delete data)

3. **Check Device Storage**
   - Low storage can slow apps
   - Free up space on device

4. **Close Background Apps**
   - Many apps running can affect performance
   - Close unused apps

### Sync Issues (Future Feature)

**Note**: Cloud sync not yet available. Local storage only.

When sync is released:
- Ensure internet connection
- Check iCloud account is signed in
- Verify iCloud Drive is enabled

## FAQ

### General Questions

#### Is Swiff free to use?

Yes! Swiff is completely free with all features included. Future premium features (like cloud sync and custom themes) may have optional pricing, but core subscription tracking will always be free.

#### Do I need to create an account?

No! Swiff works entirely on your device. No account required, no emails, no passwords. Just download and start using.

#### Where is my data stored?

All your data is stored securely on your iPhone or iPad. Nothing is sent to servers. You have complete control and privacy.

#### Can I use Swiff on multiple devices?

Currently, Swiff is single-device only. Cloud sync across devices is coming in a future update.

#### Does Swiff connect to my bank?

No. Swiff doesn't connect to banks or credit cards. You manually track subscriptions and expenses. This ensures maximum privacy and security.

### Subscription Features

#### How many subscriptions can I track?

Unlimited! Track as many as you need.

#### Can Swiff automatically detect subscriptions?

Not yet. You manually add subscriptions. We're exploring automatic detection from email receipts in future versions.

#### What if my subscription price changes?

Update the price in Swiff, and it will automatically track the change in the price history. You'll see a chart of all price changes over time.

#### Can I track annual subscriptions?

Yes! Set billing cycle to "Yearly" and Swiff tracks accordingly.

#### What about subscriptions billed quarterly or bi-annually?

Swiff supports:
- Weekly
- Bi-weekly
- Monthly
- Quarterly (every 3 months)
- Bi-annually (every 6 months)
- Annually (yearly)
- Custom intervals

### Reminders & Notifications

#### When do I get reminded?

You choose! Set reminders 1, 3, 7, 14, or 30 days before renewal. Or set custom dates.

#### Can I snooze a reminder?

Yes, in future updates. Currently, you can dismiss and it won't remind again for that billing cycle.

#### What if I miss a reminder?

Swiff sends one reminder per billing cycle at your chosen time. If missed, check the subscription detail view for next renewal date.

#### Can I change the reminder time?

Yes! Set quiet hours and preferred reminder time in Settings.

### Privacy & Security

#### Is my financial data secure?

Yes! All data stays on your device. No servers, no cloud (currently), no third parties. You can optionally encrypt backups with a password.

#### Can I lock the app with Face ID?

Coming in next update! Biometric lock and PIN protection.

#### Do you sell my data?

Never. We don't collect your data, so there's nothing to sell. Your privacy is our priority.

#### What data does Swiff collect?

None. Zero. Nada. No analytics, no tracking, no telemetry. Completely private.

### Sharing & Collaboration

#### How do shared subscriptions work?

Add a subscription, then tap "Share" to split costs with others. Swiff tracks who owes what based on your split configuration.

#### Can others see my data?

No. Sharing is one-way. You track what people owe you, but they can't access your Swiff data unless you physically show them your device.

#### Future: Will there be real-time sharing?

We're considering a feature where multiple people can collaborate on shared subscriptions. Would require accounts and cloud sync.

### Backup & Data

#### How often should I backup?

We recommend monthly backups, or before any major app update.

#### Where should I save backups?

- iCloud Drive (automatic sync across devices)
- Files app (local on device)
- Email to yourself
- Any cloud service (Dropbox, Google Drive, OneDrive)

#### Can I restore on a different device?

Yes! Create backup on Device A, transfer the .json file to Device B, and import. Your data transfers completely.

#### What if I lose my backup file?

Unfortunately, if the backup is lost and you've deleted the app, data can't be recovered. Always keep multiple backup copies!

### Export & Import

#### What format is the export?

- **JSON**: Complete data with all fields
- **CSV**: Spreadsheet format (good for Excel, Google Sheets)

#### Can I import from other apps?

Coming soon! We're adding import templates for:
- Bobby
- Rocket Money (Truebill)
- SubscriptionsTracker
- Generic CSV format

#### Can I edit the CSV and reimport?

Yes, but be careful! Editing CSV files can break data structure. Always test with a copy first.

### Technical Questions

#### What iOS version do I need?

Swiff requires iOS 16.0 or later.

#### Does it work on iPad?

Yes! Fully optimized for iPad with larger layouts.

#### Does it work on Mac?

Not yet. iOS and iPadOS only currently.

#### Is there an Apple Watch app?

Not yet, but it's on our roadmap!

#### What about widgets?

Coming soon! Home screen widgets to show upcoming renewals and monthly spending.
